# DrTests
A extendable, plugins-based UI for testing Pharo projects.

## Install

Load stable version:

```smalltalk
Metacello new
	repository: 'github://juliendelplanque/DrTests/src';
	baseline: 'DrTests';
	load
```


Load development version:

```smalltalk
Metacello new
	repository: 'github://juliendelplanque/DrTests:dev/src';
	baseline: 'DrTests';
	load
```
